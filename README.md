**Project**: LinguaFlow — AI-powered browser translation tool

- **Description**: A lightweight, polished web app for fast, privacy-friendly translations. Supports 65+ languages with auto-detection, translation history, text-to-speech, confidence scoring, and extras like formal/casual tones, alternative phrasings, romanization, and back-translation.

**Features**
- **Languages**: 65+ language support and auto-detect.
- **TTS**: Text-to-speech for source and translated text.
- **History**: Save and browse recent translations.
- **Extras**: Formal/casual registers, alternatives, explain cultural context, back-translation.
- **UI**: Responsive, accessible interface with theme toggle and quick-language chips.

**Tech stack**
- **Frontend**: HTML, CSS, JavaScript
- **Files**: index.html, style.css, app.js, plus helper modules in js (`api.js`, `languages.js`, `tts.js`, `ui.js`, `history.js`).

**Quick start**
- Open index.html in a modern browser.
